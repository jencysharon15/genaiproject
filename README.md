AI Chatbot App

A Streamlit-based AI chatbot developed using Python, LangChain, and Groq API. This project allows users to interact with an AI assistant through a clean web interface with real-time responses.

Features
Real-time AI-generated responses
Interactive chat interface
Session-based chat memory
Streamlit web deployment
Secure API key integration

Technologies Used
Python
Streamlit
LangChain
Groq API
python-dotenv

Installation
pip install -r requirements.txt
streamlit run app1.py

Deployment
Deployed using Streamlit Community Cloud through GitHub integration.

Project Purpose
Built as a practical project to explore AI application development, cloud deployment, and interactive web app design.
